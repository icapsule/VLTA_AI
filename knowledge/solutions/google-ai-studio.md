# 🛠 Google AI Studio: 帮你造 AI 的“开发工作台”

---

| 属性 | 内容 |
| :--- | :--- |
| **定位** | AI 应用开发平台 (PaaS) |
| **角色** | P1: 模型实验场 / 极速原型开发 |
| **状态** | **活跃使用 (Active)** |
| **平替 (Alternatives)** | Poe API, Vercel AI SDK, LangChain Smith |

---

## 🚀 核心特点
- **模型实验场**：免费测试 Gemini 2.x/3.x 全系列模型。
- **全栈应用生成器**：自然语言生成带数据库 (Firebase) 和用户认证的 Web 应用。
- **智能体开发底座**：开放代码执行、Google 搜索联网、函数调用 (Function Calling)。

## 🧩 典型功能
- **一键部署**：应用可直接部署到 Google Cloud Run。
- **提示词工程**：完善的提示词测试与版本管理。
- **Firebase 集成**：从想法到上线的全链路打通。

## ⚖️ 指赢官优缺点点评
- **优点**：免费配额丰厚、PaaS 级别集成力。
- **缺点**：深度绑定 Google 生态，适合开发者而非普通办公族。
