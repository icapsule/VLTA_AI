## 🚀 核心特点
- **本地优先**：所有数据和操作完全在用户掌控之中。
- **系统级控制**：通过 Accessibility API 操作本地应用（iMessage, Finder, Keynote）。
- **开源集市生态**：社区贡献 5700+ 技能插件。
- **硬件生态**：适配 Rokid 智能眼镜、Vbot 机器狗、树莓派机器人。

## 🧩 最新能力 (2026.3.7 版)
- **无损记忆 (Lossless-Claw)**：引用上下文引擎插件，解决 AI 健忘问题。
- **持久化 ACP 绑定**：系统重启后 Agent 与管理频道绑定关系不丢失。
- **模型灵活**：支持 GPT-5.4、Gemini 3.1 Flash-Lite 等。

## ⚖️ 指挥官优缺点点评
- **优点**：极致隐私安全、完全控制硬件。
- **缺点**：部署门槛高（需 Docker/API Key）、需自行维护环境。
