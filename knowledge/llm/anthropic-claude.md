# 🖋 Anthropic Claude

## 核心定位
Claude 被认为是最具“书卷气”和“逻辑性”的模型。在代码生成、长文润色和逻辑重构三个领域，Claude 3.5 Sonnet 目前被公认为行业第一。

## 技术亮点
- **Artifacts 视图**：首创了对话旁边的即时代码/网页渲染视图，极大提升了生产力。
- **编码能力**：在 Agentic Coding 场景下，其代码正确率和架构理解能力优于 GPT-4o。

## 指挥官点评
> "现在 Antigravity 或 Cursor 背后最强大的灵魂往往就是 Claude 3.5。它的逻辑非常细腻，不容易产生‘幻觉’。"
